# Attendance GUI

Simple browser-based attendance manager with export capabilities (JSON / text). Uses Bootstrap + Vite + TypeScript.

## Quick links

- App shell: [index.html](index.html)  
- Entrypoint: [src/main.ts](src/main.ts) — bootstraps UI and wires up [`exportButton`](src/elements.ts) and strings via [`StringContent.parseStrings`](src/strings.ts).  
- Grid & item UI: [`GridManager`](src/grid/GridManager.ts), [`ColItem`](src/grid/ColItem.ts) (states in [`ItemState`](src/grid/ColItem.ts)).  
- Export logic & options: [`getExportOptions`](src/export.ts), [`ExportFormat`](src/export.ts), and format handlers in [src/formats.ts](src/formats.ts).  
- Sample data: [public/json/attendanceList.json](public/json/attendanceList.json) and localization [public/json/strings.json](public/json/strings.json).  
- Dev config: [vite.config.ts](vite.config.ts)  
- Helper script to build sample lists (moved to project root): [`attendanceMaker.cjs`](attendanceMaker.cjs)  
- Package scripts: [package.json](package.json) — run the helper with `npm run create-list`

## Features

- Interactive grid of people (toggle present/absent/unspecified).  
- Export attendance as file or generated text with optional date/labels/count. Handled by [`src/formats.ts`](src/formats.ts).  
- Uses Bootstrap SCSS with aliases configured in [vite.config.ts](vite.config.ts).

## Run locally

Requirements: Node 18+ (or compatible), npm.

Install and run dev server:

```sh
npm install
npm run dev
# open http://localhost:5000
```

Build / preview:

```sh
npm run build
npm run preview
```

Create a sample attendance list:

```sh
npm run create-list
# runs: node attendanceMaker.cjs
```

## Notes for contributors

- UI strings are loaded from [`StringContent.parseStrings`](src/strings.ts) — update [public/json/strings.json](public/json/strings.json) for translations.  
- Attendance fixtures live in [public/json/attendanceList.json](public/json/attendanceList.json). Use the helper script (`attendanceMaker.cjs`) to generate lists offline; run it via `npm run create-list`.  
- SCSS entry is [src/styles/default.scss](src/styles/default.scss) which imports `_styles.scss`.

## Troubleshooting

- If Bootstrap SCSS import warnings appear, see `css.preprocessorOptions.scss.silenceDeprecations` in [vite.config.ts](vite.config.ts).  
- If DOM queries fail, check the selectors defined in [src/elements.ts](src/elements.ts).  
- If you moved the helper script, ensure `attendanceMaker.cjs` is at the project root (same folder as package.json) so `npm run create-list` works.
